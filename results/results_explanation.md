# Outcome Analysis

The federated learning simulation was executed across three simulated clients using the MNIST dataset. Model accuracy improved across communication rounds as local updates were aggregated using the Federated Averaging (FedAvg) algorithm.

Round 1 Accuracy: 83.45%  
Round 2 Accuracy: 88.08%  
Round 3 Accuracy: 89.26%  

The improvement demonstrates that collaborative training across distributed clients can progressively improve the global model without requiring centralized access to all training data. These results support the effectiveness of federated learning as a distributed training approach.
